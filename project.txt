Project: Multiplayer Trivia & Quiz Game Platform


Project Description:
Develop a multiplayer trivia and quiz game platform where users can join from their computers, participate in real-time
quiz sessions, and compete for the highest score.
The application should allow users to register, join or create game rooms, and answer timed multiple-choice questions 
that appear simultaneously for all participants in a session. 
After each round, players earn points based on their accuracy and speed, and at the end of the game, 
the system displays the leaderboard showing overall rankings.


The system should consist of a central backend server that manages players, questions, game sessions, and scores,
and a desktop application through which players interact with the platform. 
Each player should be able to view available games, join ongoing matches, and see live score updates as the game progresses.
The final product should deliver an engaging, competitive experience that tests players’ knowledge across different 
topics.
